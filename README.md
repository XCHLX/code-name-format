# code-name-format

将任意写法变量名转换为多种格式变量名，另兼有中文翻译后转换功能(需配置百度翻译 appkey,密钥) ; 支持格式 小驼峰,大驼峰,常量,下划线(小),下划线(大),分词(大),分词(小),连字符
